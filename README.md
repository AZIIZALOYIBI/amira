# Amira Chess ♔♚

موقع شطرنج فاخر احترافي مع تصميم راقي وميزات متقدمة للعب الثنائي

## المميزات الجديدة - التصميم الفاخر 🌟

### تصميم فاخر راقي
- 🏆 **ثيم ذهبي فاخر**: تصميم راقي باستخدام درجات الذهبي والألوان الداكنة الأنيقة
- ✨ **تأثيرات بصرية متقدمة**: خلفية متحركة بتأثيرات ذهبية لامعة
- 💎 **ظلال وانعكاسات فاخرة**: إضاءة وظلال احترافية على جميع العناصر
- 🎨 **تدرجات لونية راقية**: استخدام التدرجات اللونية الفاخرة في كل التفاصيل

### تجربة لعب احترافية للاعبين
- 👥 **تخصيص أسماء اللاعبين**: إدخال أسماء مخصصة للاعب الأبيض والأسود
- 🤖 **اللعب ضد الكمبيوتر**: ذكاء اصطناعي متقدم بثلاثة مستويات صعوبة (سهل، متوسط، صعب)
- 🎮 **أوضاع لعب متعددة**:
  - **لعب محلي** - لعب مع صديق على نفس الجهاز
  - **لعب ضد الكمبيوتر** - تحدي الذكاء الاصطناعي
  - **لعب عبر الإنترنت** - لعب مع أصدقاء عبر كود الدخول
- 🎮 **أوضاع زمنية متعددة**:
  - **كلاسيكي** (10 دقائق لكل لاعب)
  - **سريع** (5 دقائق لكل لاعب)
  - **خاطف** (3 دقائق لكل لاعب)
  - **غير محدود** (بدون مؤقت)
- 🏅 **شاشة إعداد اللعبة**: واجهة فاخرة لإعداد اللعبة قبل البدء
- 🎉 **احتفالات الفوز**: رسوم متحركة فاخرة عند الفوز مع كأس ذهبي

### تحسينات التصميم
- 🎯 **لوحة شطرنج فاخرة**: حدود ذهبية وتأثيرات ظل متقدمة
- ⭐ **قطع شطرنج محسّنة**: تأثيرات hover وانتقال سلسة عند التحريك
- 💫 **إضاءة ديناميكية**: تمييز المربعات المختارة بإضاءة ذهبية
- 📱 **تصميم متجاوب محسّن**: تجربة فاخرة على جميع أحجام الشاشات

## المميزات

### ميزات اللعبة الأساسية
- ✅ تطبيق كامل لقواعد الشطرنج الرسمية
- ✅ التحقق من صحة جميع الحركات
- ✅ كشف حالات الكش والكش مات

- ✅ التبييت (Castling) - جانب الملك وجانب الوزير

- 🎯 تمييز الحركات الصحيحة بإضاءة ذهبية

## كيفية الاستخدام

### التشغيل المحلي

1. قم بتحميل الملفات
2. افتح ملف `index.html` في المتصفح

أو استخدم خادم محلي:

```bash
# باستخدام Python 3
python3 -m http.server 8000

# أو باستخدام Node.js
npx http-server

# ثم افتح المتصفح على
# http://localhost:8000
```

### طريقة اللعب

1. **إعداد اللعبة**:
   - أدخل أسماء اللاعبين
   - اختر وضع اللعب (كلاسيكي، سريع، خاطف، أو غير محدود)
   - اضغط "ابدأ اللعبة"
2. **اختيار القطعة**: انقر على القطعة التي تريد تحريكها
3. **إظهار الحركات**: سيتم إظهار جميع الحركات الصحيحة بإضاءة ذهبية
4. **التحريك**: انقر على المربع المطلوب أو اسحب القطعة
5. **الحركات الخاصة**:
   - التبييت: حرك الملك خانتين نحو الرخ
   - الترقية: عند وصول البيدق للصف الأخير، اختر القطعة
6. **الفوز**: عند انتهاء اللعبة، ستظهر شاشة احتفالية فاخرة بالفائز

### الإعدادات

- **الأصوات**: تفعيل/تعطيل المؤثرات الصوتية
- **إبراز الحركات**: إظهار/إخفاء الحركات الممكنة
- **المؤقت**: اختيار وضع اللعب (كلاسيكي، سريع، خاطف، غير محدود)
- **أسماء اللاعبين**: تخصيص أسماء اللاعبين للتجربة الشخصية
