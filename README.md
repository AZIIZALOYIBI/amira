# Amira Chess ♔♚

موقع شطرنج احترافي متقدم مع واجهة مستخدم عصرية وميزات متكاملة

## المميزات

### ميزات اللعبة الأساسية
- ✅ تطبيق كامل لقواعد الشطرنج الرسمية
- ✅ التحقق من صحة جميع الحركات
- ✅ كشف حالات الكش والكش مات
- ✅ كشف حالة الطريق المسدود (Stalemate)
- ✅ نظام إدارة الأدوار بين اللاعبين

### الحركات الخاصة
- ✅ التبييت (Castling) - جانب الملك وجانب الوزير
- ✅ الأخذ بالمرور (En Passant)
- ✅ ترقية البيدق (Pawn Promotion) - مع اختيار القطعة

### واجهة المستخدم
- 🎨 تصميم عصري واحترافي
- 🌙 ألوان متناسقة وجذابة
- ✨ رسوم متحركة سلسة للحركات
- 📱 تصميم متجاوب يعمل على جميع الأجهزة
- 🔄 إمكانية قلب اللوحة
- 🎯 تمييز الحركات الصحيحة
- 📜 عرض سجل الحركات

### الميزات المتقدمة
- ⏱️ مؤقت للاعبين (قابل للتفعيل/التعطيل)
- 🔊 مؤثرات صوتية للحركات
- 🎮 السحب والإفلات (Drag & Drop)
- 📊 عرض القطع المأسورة
- 🎯 تمييز الملك في حالة الكش
- 🔍 تمييز آخر حركة

### أدوات التحكم
- 🆕 بدء لعبة جديدة
- ↩️ التراجع عن الحركة
- 🔄 إعادة تعيين اللعبة
- 🔃 قلب اللوحة
- ⚙️ إعدادات قابلة للتخصيص

## التقنيات المستخدمة

- **HTML5** - هيكل الصفحة
- **CSS3** - التصميم والرسوم المتحركة
  - CSS Grid للوحة الشطرنج
  - Flexbox للتخطيط
  - Animations & Transitions
  - Responsive Design
- **JavaScript (ES6+)** - منطق اللعبة
  - Object-Oriented Programming
  - Web Audio API للأصوات
  - Drag & Drop API
  - Event Handling

## كيفية الاستخدام

### التشغيل المحلي

1. قم بتحميل الملفات
2. افتح ملف `index.html` في المتصفح

أو استخدم خادم محلي:

```bash
# باستخدام Python 3
python3 -m http.server 8000

# أو باستخدام Node.js
npx http-server

# ثم افتح المتصفح على
# http://localhost:8000
```

### طريقة اللعب

1. **اختيار القطعة**: انقر على القطعة التي تريد تحريكها
2. **إظهار الحركات**: سيتم إظهار جميع الحركات الصحيحة
3. **التحريك**: انقر على المربع المطلوب أو اسحب القطعة
4. **الحركات الخاصة**:
   - التبييت: حرك الملك خانتين نحو الرخ
   - الترقية: عند وصول البيدق للصف الأخير، اختر القطعة

### الإعدادات

- **الأصوات**: تفعيل/تعطيل المؤثرات الصوتية
- **إبراز الحركات**: إظهار/إخفاء الحركات الممكنة
- **المؤقت**: تفعيل مؤقت 10 دقائق لكل لاعب
